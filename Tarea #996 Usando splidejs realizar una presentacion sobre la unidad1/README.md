## 🚀 Cómo Usar

1. Clona el repositorio:
    ```bash
    git clone https://github.com/SergioColli/splidejs.git
    ```
2. Navega al directorio del proyecto:
    ```bash
    cd splidejs
    ```
3. Abre el archivo `index.html` en tu navegador preferido.
